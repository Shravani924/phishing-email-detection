# 🛡️ Phishing Email Detection System

An AI-powered web application that detects whether an email is **Phishing** or **Safe** using Machine Learning.

## 🚀 Live Features
- Detects phishing emails in real time
- Simple and clean Streamlit interface
- Open access (no login required)
- Feedback & suggestion system
- About Developer and Project sections

## 🧠 Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Git & GitHub

## 💡 How to Run Locally

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   streamlit run app.py
